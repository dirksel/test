About Learn.js

The Learn.js book series is highly inspired by the lean publishing model. Each book is released early to get feedback from you so that together we can make the best books about JavaScript possible.

Updates are released regularly until each book is feature complete at v1.0.0. Help determine what it will mean for a book to be feature complete by letting us know which development tools, libraries, and programming patterns you'd be most interested in seeing covered. Email hi@learnjs.io.
